# Insurance Data Analysis

Jag, en analytiker på det påhittade försäkringsbolaget Villkor i Finstil, har fått i uppdrag att undersöka om bolagets prissättning verkar följa rimliga mönster. Villkor i Finstil vill också veta vilka faktorer som driver kostnader mest. Jag ska även undersöka om en regressionsmodell kan användas som stöd vid framtida prissättning av liknande kunder (cashing!).
